# Summary

Technology Grasp tool by Spring Boot

## Required
* Java >= 1.6
* Gradle >= 2.1
* Spring Boot Stable Ver.
* eclipse
 
## Started

```
$ gradle eclipse
$ gradle build
$ java -jar build\libs\gs-spring-boot-0.1.0.jar
```

## Structure

### Serverside
Spring Boot

### Frontside
Thymeleaf(HTML5 template engine) => http://www.thymeleaf.org/
